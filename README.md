# Linspo UI

Chart-only UI component library for React + Next.js.

## What it is

- Copy-paste chart components (shadcn-style workflow)
- Minimal, high-contrast visual language designed for dashboards
- Components focused on trends, comparisons, composition, monitoring, and correlation

## Routes

- `/` - Library landing page
- `/components` - Searchable chart component catalog
- `/components/<slug>` - Component docs with live preview + usage
- `/demo` - Full dashboard composition demo

## Included chart primitives

- `Line Chart`
- `Area Chart`
- `Bar Chart`
- `Column Chart`
- `Stacked Bars`
- `Donut Chart`
- `Pie Chart`
- `Scatter Plot`
- `Heatmap`
- `Gauge Chart`

## Run locally

```bash
npm install
npm run dev
```
